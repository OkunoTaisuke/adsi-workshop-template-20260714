## 画面詳細

### 1. ログイン画面

```
┌──────────────────────────────────┐
│         勤怠管理システム           │

│                                  │
│  ※エラー時: 赤文字でメッセージ    │
└──────────────────────────────────┘
```

### 2. ダッシュボード（打刻画面）

```
┌──────────────────────────────────────┐
│ [ナビゲーション]        田中太郎 ▼   │
├──────────────────────────────────────┤
│                                      │
│  2026年7月14日（月）                  │
│                                      │
│  状態: 勤務中                        │
│  出勤: 09:15                         │
│  休憩: 12:00 - 13:00 (60分)          │
│                                      │
│  ┌─────────┐  ┌─────────────┐       │
│  │ 退勤    │  │ 休憩開始     │       │
│  └─────────┘  └─────────────┘       │
│                                      │
└──────────────────────────────────────┘
```

打刻ボタンの表示ロジック:
- 未出勤 → 「出勤」ボタンのみ
- 出勤済み → 「退勤」「休憩開始」ボタン
- 休憩中 → 「休憩終了」ボタンのみ
- 退勤済み → ボタンなし（「本日の勤務は終了しました」）

### 3. 勤怠一覧（自分）

```
┌──────────────────────────────────────────────┐
│ 勤怠一覧          ◀ 2026年7月 ▶             │
├──────────────────────────────────────────────┤
│ 日付    | 出勤  | 退勤  | 休憩  | 勤務  |残業│
│ 7/1(火) | 09:00 | 18:00 | 1:00  | 8:00  | -  │
│ 7/2(水) | 09:30 | 19:00 | 1:00  | 8:30  |0:30│
│ 7/3(木) | 10:00 | 17:00 | 0:30  | 6:30  | -  │
│ ...                                          │
├──────────────────────────────────────────────┤
│ 合計: 勤務 15日 / 160:00 / 残業 5:30        │
└──────────────────────────────────────────────┘
```

- 各行クリックで打刻修正画面へ遷移

## 共通コンポーネント

| コンポーネント | 説明 |
|---------------|------|
| Header/Nav | ナビゲーション + ユーザー名 + ログアウト |
| MonthPicker | 年月選択（← →） |
| StatusBadge | 状態ラベル（申請中/承認/却下） |
| ConfirmDialog | 確認ダイアログ |
| ErrorMessage | バリデーションエラー表示 |
