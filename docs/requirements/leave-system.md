# 休暇申請システム — 要求仕様

## 概要

社員が有給休暇を申請し、直属の上司が承認/却下する機能。
組織管理（部署・社員登録）を前提とする。

## ユーザーと権限

| ロール | DB値 | できること |
|--------|------|-----------|
| システム管理者 | ADMIN | 部署管理、社員登録、ロール付与 |
| 上司 | MANAGER | 部下の休暇申請の承認/却下＋休暇申請の作成・キャンセル（自己承認） |
| 一般ユーザー | USER | 休暇申請の作成・キャンセル、自分の申請一覧閲覧 |

- ADMIN はシステム管理専用（部署・ユーザーの登録管理）
- MANAGER は承認権限を持つ（manager_id で紐付いた部下の申請を処理）
- MANAGER 自身の休暇申請は自動承認（申請と同時に APPROVED になる）
- 1人が複数ロールを持てる（例: ADMIN かつ MANAGER）

## 機能一覧

### F-7a: 組織管理（部署）

- ADMIN が部署を作成・参照・更新・削除できる（CRUD）
- 部署には名前を持つ

### F-7b: 社員登録

- ADMIN が社員を登録できる
- 登録項目: 名前、メールアドレス、パスワード、所属部署、上司（氏名で選択）、ロール（複数選択可）
- 上司は manager_id として Employee に保持する
- ロールは ADMIN / MANAGER / USER から複数割り当て可能

### F-7c: 休暇申請

- 一般ユーザー（USER）が休暇種別・期間・区分を指定して申請する
- 申請ステータス: PENDING → APPROVED / REJECTED / CANCELLED
- PENDING 状態の申請は本人がキャンセルできる

### F-7d: 休暇承認・却下

- 直属の上司（申請者の manager_id に該当する MANAGER ロール持ち社員）のみが承認/却下可能
- 却下理由は任意（入力なしでも却下可能）
- 承認・却下後の通知は不要（画面上で確認）

## 休暇種別

| 種別 | DB値 | 画面表示 |
|------|------|---------|
| 有給休暇 | PAID | 有給休暇 |

残日数管理はスコープ外（申請のみ記録する）。

## 休暇区分（半日休暇対応）

| 区分 | DB値 | 画面表示 |
|------|------|---------|
| 全日 | FULL_DAY | 全日 |
| 午前休 | AM_ONLY | 午前休 |
| 午後休 | PM_ONLY | 午後休 |

- 1回の申請は全日 or 半日で統一（混在不可。混在させたい場合は申請を分ける）

## ビジネスルール

### 申請ルール

- 開始日 ≤ 終了日
- 過去日の申請も可能
- 申請可能期間の上限なし
- PENDING 状態なら本人がキャンセル可能（ステータスを CANCELLED に変更）
- MANAGER ロールの社員が申請した場合は自動承認（申請時に即 APPROVED、承認者は自分自身）

### 重複申請の挙動

- 同一日に既存の PENDING 申請がある場合: 既存を自動キャンセル（CANCELLED）し、新規を PENDING で作成
- 同一日に既存の APPROVED 申請がある場合: 新規申請を受け付ける（承認されれば上書き）

### 承認・却下ルール

- 承認/却下は直属の上司（申請者の manager_id に該当する MANAGER ロール持ち社員）のみ可能
- PENDING 状態のみ承認/却下可能（処理済みなら 409 Conflict）
- 承認時に承認者 ID・承認日時を記録する

## 画面一覧

| 画面 | ロール | 機能 |
|------|--------|------|
| 部署管理 | ADMIN | 部署の CRUD |
| 社員登録 | ADMIN | 社員の新規登録（部署・上司・ロール設定） |
| 休暇申請 | USER / MANAGER | 自分の申請一覧（全件表示）＋ 新規申請フォーム＋ キャンセル。MANAGER の申請は自動承認 |
| 休暇承認 | MANAGER | 部下の申請一覧（全件表示）＋ 承認/却下ボタン |

- フィルタ: なし（全件表示）
- ページネーション: なし（50人規模で十分）

## データ保持

- 全申請（CANCELLED, REJECTED 含む）を無期限に保持する

## 非機能要件

- レスポンス: 画面操作 2秒以内
- 対応規模: 〜50人

## スコープ外

- 有給残日数管理
- 病気休暇・その他休暇の種別
- メール通知
- 申請期間の上限制御
